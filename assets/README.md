# Assets

## Required: og-image.png

Create an Open Graph image for social sharing (LinkedIn, Twitter, etc.).

**Specifications:**
- **Size:** 1200 x 630 pixels (standard OG size)
- **Format:** PNG or JPG
- **Filename:** `og-image.png`

**Suggested content:**
- Title: "Claude Memory User Edits Guide"
- Subtitle: "62% fewer errors • 100% accuracy improvement"
- Your branding/colors
- Optional: Claude/Anthropic visual reference

**Tools to create:**
- [Canva](https://canva.com) — Free templates
- [Figma](https://figma.com) — More control
- [OG Image Generator](https://og-image.vercel.app) — Quick text-based

**Test your OG image:**
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)

---

Once created, place `og-image.png` in this folder.
