# Template: Content & Writing Projects

*Starter memory edits for content creation and writing projects.*

---

## Core Template (5 edits)

Copy and customize these:

```
1. "Tone: [Professional/Casual/Academic/Conversational]"
2. "Audience: [Job titles, experience level, industry]"
3. "Format: [Blog posts/Technical docs/Marketing copy/Social media]"
4. "Constraints: [Word limits, style guide, SEO requirements]"
5. "Brand: [Company name, positioning, voice guidelines]"
```

---

## Filled Example

```
1. "Tone: Professional but approachable, avoid jargon" (51 chars)
2. "Audience: B2B SaaS founders, technical background" (51 chars)

4. "Constraints: SEO-optimized, Hemingway grade 8, no clickbait" (61 chars)
```

## Content Type Additions

### Blog/SEO Content
```
"Blog length: 1500-2500 words, H2 every 300 words" (49 chars)
"SEO: Include primary keyword in H1, first 100 words" (52 chars)
"Links: 3-5 internal, 2-3 external authority sources" (52 chars)
```

### Social Media
```
"LinkedIn: Hook in first line, no hashtag spam (max 3)" (54 chars)
"Twitter/X: Punchy, one idea per tweet, thread for depth" (56 chars)
"Format: Short paragraphs, use line breaks for readability" (58 chars)
```

### Technical Documentation
```
"Docs: Task-oriented, not feature-oriented" (42 chars)
"Code examples: Working, tested, with comments" (45 chars)
"Structure: Overview → Steps → Troubleshooting" (46 chars)
```

### Email/Marketing Copy
```
"Subject lines: Under 50 chars, curiosity or benefit" (52 chars)
"Email length: 150-300 words for newsletters" (43 chars)
"CTA: One clear action per email, button preferred" (50 chars)
```

---

## Voice & Style

```
"Voice: We, not I. Active voice. Present tense." (46 chars)
"Avoid: Corporate speak, buzzwords, 'leverage', 'synergy'" (57 chars)
"Sentence length: Average 15-20 words, vary for rhythm" (54 chars)
```
